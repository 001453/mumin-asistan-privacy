# Mümin Asistan - Gizlilik Politikası

Bu klasör Google Play Store için Gizlilik Politikası sayfasıdır.

## GitHub Pages ile Yayınlama

### 1. GitHub'da yeni repo oluşturun
- https://github.com/new adresine gidin
- Repository adı: `mumin-asistan-privacy` (veya başka bir isim)
- Public seçin
- "Create repository" tıklayın

### 2. Dosyaları yükleyin
- "uploading an existing file" veya "Add file" → "Upload files"
- `index.html` dosyasını sürükleyip bırakın
- "Commit changes" tıklayın

### 3. GitHub Pages'i etkinleştirin
- Repo sayfasında **Settings** → **Pages**
- "Source" altında **Deploy from a branch** seçin
- Branch: **main**, Folder: **/ (root)**
- **Save** tıklayın

### 4. URL'inizi alın
Birkaç dakika sonra sayfa yayında olacak:
```
https://KULLANICI_ADINIZ.github.io/mumin-asistan-privacy/
```

Bu URL'i Google Play Console → App content → Privacy policy alanına yapıştırın.
